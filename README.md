# Multi-Market Live Tracker

Dashboard monitoring market real-time yang mencakup Forex, Crypto, dan Indeks Saham. Proyek ini dibuat untuk memfasilitasi analisis teknikal cepat (seperti XAUUSD) melalui web browser.

## Fitur Utama
- **Live Data**: Harga terupdate secara otomatis.
- **Multi-Asset**: Mendukung tab untuk memisahkan kategori aset.
- **Responsive**: Bisa dibuka melalui Mobile (Replit) maupun Desktop.
- **Dark Mode**: Desain estetik yang ramah mata.

## Cara Menggunakan
1. Clone repository ini.
2. Buka `index.html` di browser kamu.
3. Atau aktifkan **GitHub Pages** untuk menghosting secara gratis.
